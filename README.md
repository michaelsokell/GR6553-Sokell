Within these files are different meterological/forecasting tools using python code to analysis the meteorlogical environment that created the historical May 31, 1985 tornado outbreak in the Northeast United States and Ontario, Canada. 300mb analysis,500 Relative Vorticity, 700mb Relative Humidity environment of the F5 Wheatland Tornado, 850mb analysis, Upper Air Soundings, as well as ERA5 Surface Analysis to give all the tools needed to disect the atmosphere. Needed files for this are three ERA5 files in May 30,31, and June 1st which you will need to unzip, but apply them in the code for the appropriate day and they should work! Enjoy!
